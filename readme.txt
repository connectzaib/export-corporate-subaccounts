Export MemberPress Corporate Account sub-users to CSV and view detailed dashboard reports.

== Description ==

This plugin enhances MemberPress Corporate Accounts by providing a detailed reporting dashboard and easy CSV export functionality for sub-accounts.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/export-subaccounts` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the 'Corporate Reports' menu item in the sidebar to view reports.
